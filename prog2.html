Connaissances

Théoriques (quels sont les notions que vous avez retenues)?

Modéliser une base de données en fonction de besoins spécifiques. Nous avons exploiter cette base de données à l’aide phpmyadmin ainsi qu’à utiliser un langage d’interaction permettant d’obtenir et de modifier par programmation les données. En continuant notre apprentissage de la programmation serveur, Nous apprenions les diverses techniques utilisées pour créer des pages web dynamiques. Nous avons appris des librairies de base pour créer des pages web permettant l’affichage d’informations dynamiques contenues dans des bases de données. Nous avons sommairement vue le concepts de base de la programmation orientée objet.

Pratiques (quels sont les logiciels que vous avez appris)?

  1. phpmyadmin
  2. MySQL

Conditionnelles (quels sont les projets que vous avez réalisés)?

L'épreuve synthèse portaient sur un site de vente de voitures usagées dont nous devions faire la base de données.
