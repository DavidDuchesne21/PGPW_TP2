Connaissances

Théoriques (quels sont les notions que vous avez retenues)?

La programmation orienté-object avec PHP: le développement d'architecture de requête HTTP à l'aide des concepts de classes, d'objects et d'utilisation de fonctions préféginies avec l'aide d'un language de programmation orientée-object côté serveur.

Pratiques (quels sont les logiciels que vous avez appris)?

  • Modéliser et programmer des classes à l'aide d'un langage de programmation web côté-serveur
  • Développer un site Web dynamique suivant un patron de conception orientée-objet
  • Développer un site Web dynamique avec une interface administrative suivant un patron de conception orientée-objet

Conditionnelles (quels sont les projets que vous avez réalisés)?

  • Créer un "agenda" de gestion d'activitées à l'aide de données enregistré dans une base de donnée
  • Créer un formulaire de téléversement d'images via un dossier uploads
  • Créer un site web de recherche dans une banque de livres à l'aide de mots clé
